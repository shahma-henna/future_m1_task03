# Resume Screening and Candidate Ranking System

## 📌 Project Overview
This project is a Machine Learning-based Resume Screening and Candidate Ranking System developed using Python. The system automatically analyzes resumes, matches candidate skills with the required job role, and ranks candidates based on their match percentage. It helps recruiters reduce manual screening time and improve the hiring process.

---

## 🎯 Objectives
- Automate resume screening.
- Match candidate skills with job requirements.
- Rank candidates based on similarity.
- Visualize the top candidate match percentages.
- Reduce manual effort in recruitment.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- VS Code

---

## 📂 Dataset
The project uses a CSV dataset containing:
- Candidate ID
- Resume Text
- Skills
- Job Role

---

## ⚙️ Features
- Resume text preprocessing
- Skill extraction
- Candidate ranking
- Match percentage calculation
- Data visualization
- Automatic candidate comparison

---

## 📊 Project Workflow
1. Load the dataset.
2. Clean and preprocess resume data.
3. Extract important skills.
4. Compare resumes with the job description.
5. Calculate match percentage.
6. Rank candidates.
7. Display the Top 10 candidates using a bar chart.

---

## 📈 Output
The system generates a bar chart showing the Top 10 Candidate Match Percentages, helping recruiters quickly identify the most suitable candidates.
